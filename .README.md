# CourseForge

A full-stack e-learning platform built with Next.js 14, Prisma, and Tailwind. 
Students can browse courses, enroll, and track progress. Instructors can create courses and lessons.
 
## ✨ Features

- **Auth**: Email/Password with JWT + httpOnly cookies
- **Role-based Access**: STUDENT, INSTRUCTOR, ADMIN
- **Courses**: Create, browse, and view courses
- **Lessons**: Markdown/Youtube content with ordering
- **Enrollments**: One-click enroll/unenroll
- **Progress Tracking**: Mark lessons complete + % progress bar
- **UI**: Tailwind CSS + shadcn/ui with Nova theme + Dark mode
- **Server Actions**: Type-safe mutations, no API routes needed
